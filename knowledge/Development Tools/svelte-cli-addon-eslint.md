# Svelte CLI Add-on eslint

## Overview

Installs ESLint with `eslint-plugin-svelte`, creates `eslint.config.js`, and updates VS Code settings. Integrates with TypeScript and Prettier if present. <https://raw.githubusercontent.com/sveltejs/cli/main/documentation/docs/30-add-ons/10-eslint.md>

## Usage

```bash
npx sv add eslint
```
