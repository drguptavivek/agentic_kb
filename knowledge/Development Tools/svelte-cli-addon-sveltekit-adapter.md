# Svelte CLI Add-on sveltekit-adapter

## Overview

Installs and configures an official SvelteKit adapter in `svelte.config.js`. <https://raw.githubusercontent.com/sveltejs/cli/main/documentation/docs/30-add-ons/45-sveltekit-adapter.md>

## Usage

```bash
npx sv add sveltekit-adapter
```

## Options

- `adapter`: `auto`, `node`, `static`, `vercel`, `cloudflare`, `netlify`
- `cfTarget`: `workers` or `pages` (Cloudflare only) <https://raw.githubusercontent.com/sveltejs/cli/main/documentation/docs/30-add-ons/45-sveltekit-adapter.md>
