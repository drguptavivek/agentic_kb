# Svelte CLI Add-on lucia

## Overview

Adds Lucia-based auth setup for SvelteKit with Drizzle integration and optional demo pages. <https://raw.githubusercontent.com/sveltejs/cli/main/documentation/docs/30-add-ons/15-lucia.md>

## Usage

```bash
npx sv add lucia
```

## Options

- `demo:yes` to add demo registration/login pages. <https://raw.githubusercontent.com/sveltejs/cli/main/documentation/docs/30-add-ons/15-lucia.md>
