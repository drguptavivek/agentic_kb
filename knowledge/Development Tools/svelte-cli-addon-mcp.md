# Svelte CLI Add-on mcp

## Overview

Adds Svelte MCP configuration for local/remote setup and a README for agents. <https://raw.githubusercontent.com/sveltejs/cli/main/documentation/docs/30-add-ons/17-mcp.md>

## Usage

```bash
npx sv add mcp
```

## Options

- `ide`: IDE name (e.g., `cursor`, `vscode`, `claude-code`)
- `setup`: `local` or `remote` <https://raw.githubusercontent.com/sveltejs/cli/main/documentation/docs/30-add-ons/17-mcp.md>
