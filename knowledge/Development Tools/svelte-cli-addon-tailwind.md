# Svelte CLI Add-on tailwindcss

## Overview

Adds Tailwind setup (including Vite plugin) and updates layout/app CSS. Integrates with Prettier if present. <https://raw.githubusercontent.com/sveltejs/cli/main/documentation/docs/30-add-ons/50-tailwind.md>

## Usage

```bash
npx sv add tailwindcss
```

## Options

- `plugins`: `typography` or `forms` (`@tailwindcss/typography`, `@tailwindcss/forms`) <https://raw.githubusercontent.com/sveltejs/cli/main/documentation/docs/30-add-ons/50-tailwind.md>
