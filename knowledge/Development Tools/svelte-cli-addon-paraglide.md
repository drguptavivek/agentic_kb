# Svelte CLI Add-on paraglide

## Overview

Adds Paraglide (Inlang) i18n with tree-shakable message functions, plus hooks and app.html updates. <https://raw.githubusercontent.com/sveltejs/cli/main/documentation/docs/30-add-ons/25-paraglide.md>

## Usage

```bash
npx sv add paraglide
```

## Options

- `languageTags`: comma-separated BCP-47 tags
- `demo:yes` to include a demo page <https://raw.githubusercontent.com/sveltejs/cli/main/documentation/docs/30-add-ons/25-paraglide.md>
