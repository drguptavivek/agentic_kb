# Svelte CLI sv add

## Overview

`sv add` augments an existing project with official add-ons. <https://raw.githubusercontent.com/sveltejs/cli/main/documentation/docs/20-commands/20-sv-add.md>

## Usage

```bash
npx sv add
npx sv add [add-ons]
```

## Options

- `-C, --cwd <path>`: project root
- `--no-git-check`: skip dirty-check prompt
- `--install <npm|pnpm|yarn|bun|deno>` / `--no-install` <https://raw.githubusercontent.com/sveltejs/cli/main/documentation/docs/20-commands/20-sv-add.md>

## Official Add-ons

`devtools-json`, `drizzle`, `eslint`, `lucia`, `mcp`, `mdsvex`, `paraglide`, `playwright`, `prettier`, `storybook`, `sveltekit-adapter`, `tailwindcss`, `vitest`. <https://raw.githubusercontent.com/sveltejs/cli/main/documentation/docs/20-commands/20-sv-add.md>
