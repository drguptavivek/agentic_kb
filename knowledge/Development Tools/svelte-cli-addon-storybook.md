# Svelte CLI Add-on storybook

## Overview

Runs `npx storybook init` and configures Storybook for SvelteKit or Svelte+Vite. <https://raw.githubusercontent.com/sveltejs/cli/main/documentation/docs/30-add-ons/40-storybook.md>

## Usage

```bash
npx sv add storybook
```
