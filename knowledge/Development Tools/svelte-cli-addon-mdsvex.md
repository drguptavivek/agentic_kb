# Svelte CLI Add-on mdsvex

## Overview

Adds mdsvex and configures it in `svelte.config.js` for Markdown-in-Svelte workflows. <https://raw.githubusercontent.com/sveltejs/cli/main/documentation/docs/30-add-ons/20-mdsvex.md>

## Usage

```bash
npx sv add mdsvex
```
